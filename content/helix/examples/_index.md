Browse ready-to-deploy agent templates at [deploy.helix.ml/agents](https://deploy.helix.ml/agents).

## Example Agents

- **Customer Support** - Answer questions using your documentation and knowledge base
- **Code Assistant** - Help with code review, debugging, and generation
- **Data Analyst** - Query databases and generate reports
- **Content Writer** - Draft emails, blog posts, and marketing copy
- **Research Assistant** - Summarize documents and extract insights

Each template includes pre-configured skills, knowledge sources, and system prompts that you can customize for your use case.
